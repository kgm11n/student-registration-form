# Student Registration Form

A comprehensive, fully-functional student registration form built with HTML, CSS, and JavaScript. This form includes validation, responsive design, and a clean user interface.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Form Validation**: Real-time and on-submit validation with clear error messages
- **Multiple Sections**: Organized into logical sections:
  - Personal Information
  - Academic Information
  - Contact Information
  - Additional Information
- **Age Verification**: Validates that students are at least 18 years old
- **GPA Validation**: Ensures GPA is within valid range (0-4.0)
- **Email Validation**: Validates email format
- **Phone Validation**: Validates phone number format
- **Success Message**: Displays confirmation after successful submission
- **Clean UI**: Modern design with gradient background and smooth animations

## Form Fields

### Personal Information
- First Name (required)
- Last Name (required)
- Email Address (required)
- Phone Number (optional)
- Date of Birth (required)
- Gender (required)

### Academic Information
- Student ID (required)
- Program of Study (required)
- Year of Study (required)
- Current GPA (optional)

### Contact Information
- Street Address (required)
- City (required)
- State/Province (required)
- Zip/Postal Code (required)
- Country (required)

### Additional Information
- Areas of Interest (checkboxes)
- Additional Comments (optional)
- Terms & Conditions (required)

## How to Use

1. Clone the repository:
```bash
git clone https://github.com/kgm11n/student-registration-form.git
```

2. Open `index.html` in your web browser

3. Fill out the form with your information

4. Click "Register" to submit the form
